05 — Components
Buttons + Elements
Button Rules
  • Pill shape (999px radius) — always
  • Uppercase, 12px, 0.1em tracking
  • Padding: 14px top/bottom, 28px sides
  • Max one accent button per page
  • Hover: opacity 0.85 — no color shift, no scale
Cards
  • 1px border, rgba(45,45,45,0.12)
  • 12px corner radius
  • No shadow — ever
  • 24px internal padding
Callout Lines
  • 1px stroke, charcoal #2D2D2D
  • Straight lines only — no curves, no decorative arrows
  • Snap to grid, 90 or 45 degree angles
Icons
  • Outline only, 1.5px stroke
  • Charcoal color only
  • Never filled, never colored, never illustrated

07 — Visual DNA
Image + Composition Rules
Product Photography
  • Pure white background
  • Soft grounded shadow
  • Three-quarter angle, eye level
  • Soft studio light
  • Shallow depth of field
  • Clean edges, no grain
  • Apple-like minimal precision
Product Photography — Never
  • Props (unless lifestyle category)
  • Textured backgrounds
  • Glossy or 3D effects
  • Filters or color grading
  • Busy compositions
  • Text overlay on product
Infographics
  • 1px charcoal callout lines
  • Perfect grid alignment
  • Uppercase micro-labels
  • No decorative elements
  • Minimal, clean data viz
Lifestyle
  • Natural, confident, unstaged
  • Natural or soft outdoor light
  • Real people, not overly styled
  • Never overly posed
  • Never stock-photo compositions

09 — Usage Guidelines
How to Use This System
01

Paste Universal Prompt

Start every AI session by pasting the Universal Prompt block from the Master Prompt file. This sets brand constraints.

02

Add Task Prompt

Paste the specific task prompt (product shot, landing page, email, deck, etc.) for format-specific rules.

03

Give Your Direction

Add your specific creative instructions. The layered approach keeps every AI output on-brand.

File Format Purpose
tokens.yaml YAML Machine-readable brand tokens for AI tools and code pipelines
master-prompt.md Markdown Copy/paste prompt blocks for ChatGPT, Claude, Midjourney, etc.
index.html HTML Visual spec page for humans — the page you are reading now
When to Use tokens.yaml
  • Paste relevant sections into AI context windows before giving instructions
  • Feed into design-to-code tools (Figma plugins, v0, Bolt, etc.)
  • Use as the source of truth for programmatic design decisions
  • Reference exact hex values, spacing scales, and type specs
When to Use master-prompt.md
  • Paste Universal Prompt at the start of every AI design session
  • Add task-specific prompt for the asset type you are creating
  • Use for image generation (Midjourney, DALL-E, Flux, etc.)
  • Use for copy generation (ChatGPT, Claude, etc.)

12 — AI Master Prompt
Copy + Paste Into Any AI Tool
You are designing for DeadSoxy, a product-first premium essentials brand built on engineered comfort and precision construction. BRAND: Confident, restrained, intelligent, engineered. Never hype-driven, never loud, never gimmicky. Prioritize clarity, craftsmanship, and subtle performance advantages (like TrueStay™ grip) — not trendiness or decoration. COLORS: - Charcoal #2D2D2D — primary (min 50%) - Magenta #D500F8 — accent (max 20%) - Gold #FFBB3B — rare accent (max 8%) - White #FFFFFF — primary background TYPOGRAPHY: - Gotham Black (Inter 900/Helvetica Neue Black fallback) — headlines (H1, H2) - Gotham Book (Inter/Helvetica Neue fallback) — body, UI, subheads, labels - Mrs Eaves Roman (Georgia fallback) — editorial only, max 1 per layout - Headlines: uppercase, weight 900, tight tracking - Body: sentence case, 16px, 1.6 line height - Labels: uppercase, 12px, wide letter spacing LAYOUT: - 8pt grid: 4, 8, 16, 24, 32, 48, 64, 96px - 12px corner radius - Generous whitespace - Perfect alignment NEVER: - Gradients, drop shadows on text, lens flares - Script fonts, decorative fonts - Busy patterns, textured backgrounds - Neon colors, over-saturated palettes - Stock-photo compositions - Hype words: "ultimate", "next level", "game changer"
